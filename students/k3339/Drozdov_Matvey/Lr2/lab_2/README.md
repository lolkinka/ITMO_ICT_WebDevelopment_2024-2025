# Описание проекта

Система позволяет пользователям:
- просматривать доступные туры
- регистрироваться и входить в систему
- бронировать туры
- отменять свои бронирования
- оставлять отзывы о забронированных турах
- просматривать статистику проданных туров по странам
- искать туры по названию, агентству и стране

Администратор через Django Admin может:
- создавать/редактировать/удалять туры и тур-агентства
- подтверждать бронирования

# Структура

## Модели

TourAgency:
- name
- description

Tour:
- title
- agency (ForeignKey)
- description
- country
- start_date / end_date
- price
- payment_terms

Reservation:
- user
- tour
- created_at
- is_confirmed

Review:
- user
- tour
- created_at
- rating
- text

## Функционал:

Просмотр туров:\
Туры отображаются списком, есть поиск и пагинация.

Страница тура: 
- просмотр данных тура
- бронирование (только для авторизованных)
- оставление отзывов (только после бронирования)
- пагинированный список отзывов.

Мои бронирования:\
Пользователь видит свои брони и может отменить их.

Статистика:\
Таблица проданных туров по странам + пагинация.

Регистрация, вход, выход:
Используется встроенная Django-аутентификация.
